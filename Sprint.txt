Weather App Project Breakdown

Level 1: Basic (Beginner-Friendly)
Get current weather for a city
Display temperature, weather condition, and humidity
Output results as plain text

Level 2: Intermediate

Allow searching weather for multiple cities
Save results to a text file for later reference
Show additional info: wind speed, sunrise/sunset time

Level 3: Advanced

Fetch 5-day weather forecast for cities
Allow user to input city names via a file and output results to another file
Include alerts for extreme weather (temperature thresholds, storms)
